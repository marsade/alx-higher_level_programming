Usage
=====
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 3, 5]
>>> my_list.print_sorted()
[1, 3, 4, 5]

Empty list
========
>>> MyList = __import__('1-my_list').MyList
>>> a = []
>>> my_list = MyList(a)
>>> my_list.print_sorted()
[]

Neg numbers
=========
>>> MyList = __import__('1-my_list').MyList
>>> my_list.append(-9)
>>> my_list.append(-10)
>>> my_list.append(2)
>>> my_list.append(6)
>>> my_list.print_sorted()
[-10, -9, 2, 6]

Same numbers
======
>>> my_list2 = MyList()
>>> my_list2.append(1)
>>> my_list2.append(1)
>>> my_list2.append(1)
>>> my_list2.append(1)
>>> my_list2.print_sorted()
[1, 1, 1, 1]
